~Pig Game~
A game of pig where a human player plays against a simple computer which can only 
roll the dice once per turn.

Prerequesites:

VM Argument
--module-path "${eclipse_home}/javafx-sdk-14.0.2.1/lib" --add-modules javafx.controls,javafx.fxml

Technologies Used:
-Java
-JavaFX
-Eclipse
